# Generate UI Tests

Use this command to scaffold Playwright-based UI tests for a component or flow.

## Inputs
- Target page or route
- Expected user flow
- Key assertions

## Output
- Page object file
- Flow test file
- Minimal fixture or config if needed

## Workflow
1. Inspect the target UI surface.
2. Create or update a page object.
3. Add a reusable flow test.
4. Attach assertions for visible state and error handling.
