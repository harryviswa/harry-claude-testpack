# Claude Automation Setup

This repository includes a complete Claude automation scaffold for UI and API testing.

## Commands
- /add-test
- /generate-ui-test
- /generate-api-test
- /generate-report

## Skills
- ui-navigation
- api-testing
- test-data-setup

## Hooks
- automation-lifecycle
- reporting/report-generator

## Default workflow
1. Run tests.
2. Invoke the reporting hook.
3. Generate a summary report in artifacts/automation-report.md.
